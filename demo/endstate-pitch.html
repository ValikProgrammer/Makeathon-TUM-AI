The problem

German law killed B2B outbound.

§7 UWG
Cold calls need consent. AI voice calls need explicit consent — always.
<1%
Cold email response rate. LinkedIn and WhatsApp follow the same curve — and violate ToS.
300k €
Max. fine per violation. One careless campaign ends the company.
Meanwhile, buyers publish intent constantly — openings, funding, job posts, permits — that no one converts into timely, legal outreach.
Our solution

Three agents. Each one ruthlessly scoped.

Postal is §7-free. The prospect opens the door. Then three specialists do exactly one job each.

Jack · Scout

Hunts public buying signals.

"Who's about to need this — and why?"

  • Sources — KfW funding, building permits, public tenders, expansion press
  • Scores — 6 weighted dimensions, 0–100, with hard ICP gate (no commercial kitchens)
  • Picks the hook — Simplify · Scale · Optimize · Circular, by decision-maker role
  • Triggers — personalised postcard with QR to a hook-tuned landing page
Kate · Caller

Dials only when invited.

"Tell me about your project — I'll suggest the fit."

  • Triggered — by the QR-scan + consent on the landing page (UWG §7 satisfied)
  • Discloses — AI in the first sentence (EU AI Act Art. 50)
  • Discovers + configures — Leader / Profi / Top Feature mix in 4 minutes
  • Hands off — qualified envelope written into Twin → Otto fires
Otto · Closer

Builds the proposal. Sends it.

"Reads the call, prices the bundle, hits send."

  • Reads — full Twin row (bundle mix, term, contact channel)
  • Computes — €/month × term, with the BSH Leader/Profi/Top Feature rates
  • Drafts + sends — tailored email body via the Outlook integration
  • Updates — stage → "offered", offer_sent_at → now
Signal → Postcard → QR + Consent → Voice qualification → Offer email

Stack
· Built on HappyRobot

No glue code we didn't have to write.

What HappyRobot gives us
Twin · managed Postgres
One leads table. All three agents read + write via Read / Write nodes. Cockpit reads via REST.
Outbound Voice Agent
Kate's call. STT → LLM → TTS pipeline, @variable interpolation, AI Act Art. 50 disclosure.
Native Contacts + Memories
Auto-dedup on phone/email. Per-call interaction history. No CRM glue.
AI Extract · strict JSON Schema
Kate's transcript → 16 typed fields with additionalProperties: false. No regex, no parsing.
Webhook trigger + POST
Bearer-auth endpoints kick agents off. Outbound POSTs back to the cockpit close the loop.
Outlook integration
Otto sends the proposal via the org's authorised mailbox — no SendGrid setup.
Custom Code (Python)
Otto's bundle math: leader×42 + profi×58 + top×80. 18 lines.
Runs · transcripts · audit
Full execution timeline per call. Latency breakdown for STT/LLM/TTS.
